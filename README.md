# Landing Page Practice

A responsive landing page built using HTML5 and CSS3 as a practice exercise to strengthen layout, styling, and structural fundamentals in frontend development.

## Features

- Semantic HTML5 structure
- Responsive layout
- Flexbox-based alignment
- Clean and organized CSS
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- CSS Flexbox

## Project Structure

```
├── images
├── index.html
├── README.md
└── style.css
```

## What I Practiced

- Structuring pages using semantic HTML elements
- Building layouts using Flexbox
- Managing spacing and alignment
- Writing maintainable CSS
- Applying responsive design principles

## How to Run

Open `index.html` in any modern web browser.
